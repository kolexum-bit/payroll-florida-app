# Payroll Florida App

A simple Florida payroll web app for small teams.

## Run locally
1. Install dependencies:
   - `pip install -r requirements.txt`
2. Start the app:
   - `uvicorn app.main:app --reload`
3. Open:
   - `http://127.0.0.1:8000/company`

## Main pages
- Company
- Employees
- Monthly Payroll
- Pay Stubs
- Reports
